# 郑太阳个人作品集网站结构文档

## 1. 项目定位

网站类型：个人作品集网站。

网站核心定位：面向 UI 设计、数据可视化大屏、智能终端界面、移动端产品设计、AIGC 影像制作、手绘与视觉设计的个人作品集官网。整体气质应更接近专业设计师/创意总监型作品官网，而不是普通个人主页。

来源基础：基于站酷主页 `https://www.zcool.com.cn/u/13555989` 的公开信息整理。作者名为“郑太阳”，所在地为山东青岛，站酷作品共 17 组。页面作品显示近期重点集中在综合能源大数据管理平台、智慧园区统一管理平台、智慧校园统一管理平台、智能触控终端、智慧园区 APP 等 UI/数据可视化方向；早期作品包含电商 APP、直播移动端界面、图标、手绘图标、网页、专题页、banner、版式排版、人物海报等。

网站对外表达建议：

“郑太阳是一位兼具 UI 产品设计、数据可视化界面、AIGC 影像、手绘表达与视觉设计能力的复合型设计师，擅长将复杂业务系统转译为清晰、高级、可落地的数字体验。”

## 2. 目标用户

1. 企业客户：需要数据平台、智慧园区、能源、校园、智能终端、APP 等界面设计服务。
2. 招聘方/合作方：希望快速判断设计师的专业能力、项目完整度和审美成熟度。
3. 创意合作方：关注 AIGC 影像、视觉海报、手绘和品牌视觉表达能力。

## 3. 网站结构

建议采用三大核心页面，并允许后续扩展详情页：

1. 首页 `/`
2. 作品展示 `/work`
3. 作品详情 `/work/[slug]`
4. 个人介绍 `/about`
5. 联系合作 `/contact`，可作为首页和个人介绍页底部模块，也可独立页面

导航建议：

`首页 / 作品 / 关于 / 联系`

## 4. 首页结构

### 4.1 首屏 Hero

模块作用：建立第一印象，快速说明作者是谁、擅长什么、作品气质是什么。

适合内容：

- 作者姓名：郑太阳
- 身份标签：UI Designer / AIGC Visual Creator / Digital Experience Designer
- 核心能力：UI 设计、数据可视化、智能终端、移动端产品、AIGC 影像、手绘、视觉设计
- 一句主张：将复杂系统与视觉表达转化为清晰、有质感、可落地的数字体验
- 主按钮：查看作品
- 次按钮：联系合作

### 4.2 精选作品

模块作用：优先展示最能代表当前专业方向的项目，让访问者不用翻完整列表也能判断能力。

适合放置：

- 综合能源大数据管理平台

- 智慧校园统一管理平台
- 智能触控终端
- 智慧园区 APP

展示方式：

- 2 个大尺寸主推项目 + 3 个中尺寸项目
- 每个项目显示：封面、项目名、类型标签、简短说明、年份
- 推荐标签：数据可视化、互动大屏、智慧园区、智能终端、APP UI

### 4.3 能力矩阵

模块作用：把作品背后的能力讲清楚，避免网站只像图片陈列。

适合内容：

- UI/UX Design：复杂业务系统、移动端 APP、智能设备界面

- Illustration & Icon：手绘图标、写实图标、角色/物件绘制
- Visual Design：网页、专题页、banner、版式、海报

### 4.4 项目分类入口

模块作用：让用户按能力方向浏览，不被时间顺序限制。

建议分类：

- 数据可视化与智慧系统
- APP 与移动端界面
- 智能终端界面
- 图标与手绘
- 视觉海报与版式
- 网页与专题页
- AIGC 影像实验

### 4.5 关于摘要

模块作用：在首页建立人的可信度，而不只展示作品。

适合内容：

- 青岛设计师
- 站酷十年以上创作积累
- 跨 UI、视觉、手绘、AIGC 的综合能力
- 服务方向：企业级界面、智慧系统、品牌视觉、创意影像

### 4.6 联系合作

模块作用：将浏览转化为合作咨询。

适合内容：

- 微信/电话
- 邮箱
- 所在城市：青岛，可支持远程协作
- 合作类型：UI 设计、可视化大屏、APP 设计、AIGC 影像、视觉设计

## 6. 作品详情页结构

适合内容：

- 项目名称
- 行业/类型
- 年份
- 关键词
- 大尺寸封面图

作用：帮助客户理解设计价值。

## 8. 视觉风格建议

整体关键词：专业、高级、克制、清晰、数字感、设计师气质。

### 8.1 色彩

推荐方案：

- 主背景：近黑色 `#0B0D10` 或深灰 `#111318`
- 内容背景：白色/浅灰，用于作品详情阅读区
- 强调色：冷青蓝 `#40D8FF` 或电光蓝 `#357CFF`
- 辅助色：银灰、雾白、低饱和蓝绿

注意：不要做成过度炫技的赛博风，也不要满屏渐变。首页可以有数字感，详情页要保持阅读舒适。

### 8.2 字体

中文建议：

- 思源黑体 / HarmonyOS Sans / 阿里巴巴普惠体

英文建议：

- Inter / Neue Haas Grotesk / Helvetica Now

排版建议：

- 首页标题大而克制
- 作品说明文字短句化
- 详情页使用清晰网格和足够留白

### 8.3 版式

推荐：

- 首页采用大图首屏 + 不对称作品网格
- 作品页采用筛选 + masonry 或 3 列卡片
- 详情页采用杂志式长页：大图、文字说明、界面截图交替
- 关于页采用左右分栏：个人介绍 + 能力清单

避免：

- 普通模板感卡片堆叠
- 大面积廉价渐变
- 花哨动效干扰作品
- 过多装饰性文字

## 9. 交互动效建议

1. 页面进入：轻微淡入和上移，时长 300ms 左右。
2. 作品卡片 hover：封面轻微放大，标题和标签浮现。
3. 筛选切换：平滑重排，不要闪烁。
4. 详情页图片：滚动时轻微视差，但不要影响阅读。
5. 导航：滚动后变为半透明玻璃质感。

## 10. 素材准备建议

开发前建议准备：

- 作者头像或个人形象照

- 手绘/图标作品高清图

- 联系方式
